# mini-shawn

mini-shawn は、河上の経験・知識・判断基準を言語化し、採用・組織・人事業務の企画/推進/運営を支援するための実験用プロジェクトです。

このリポジトリでは、以下を整理・公開していきます。

- 採用・組織・人事に関する知識
- 実務で使えるテンプレートやプレイブック
- AIエージェントが参照しやすい判断ルール
- Claude Code などで再利用できるエージェント定義

## このプロジェクトで目指すこと

- 実務知見を再利用可能な形で構造化する
- AIエージェントが読みやすい形で蓄積する
- 他の人が導入して業務支援に使える状態にする
- ドキュメント公開と運用資産管理を両立する

## 主なユースケース

- 求人票の改善
- 採用戦略の整理
- カジュアル面談・面接設計
- Value設計と浸透施策整理
- 採用オペレーション改善
- 人事関連の論点整理とたたき台作成

## ディレクトリの考え方

- `docs/` : 人が読むための公開ドキュメント
- `knowledge/` : 実務知見の本体
- `agents/` : エージェントの役割定義
- `prompts/` : タスク実行用の指示
- `evals/` : 品質検証用の資産

## 注意

このリポジトリの内容は判断支援を目的としたものです。実際の採用・組織・労務・法務判断では、必要に応じて専門家確認を行ってください。
